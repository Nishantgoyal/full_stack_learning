# Restful Routing

| name | url | verb | description |
| ---- | --- | ---- | ----------- |
| INDEX | /dogs | GET | Display a list of all dogs |
| NEW | /dogs/new | GET | displays a form to make a new dog |
| CREATE | /dogs | POST | Add a new dog to DB |
| SHOW | /dogs/:id | GET | Shows info about one dog |
