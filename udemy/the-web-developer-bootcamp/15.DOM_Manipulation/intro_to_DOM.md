# Introduction to DOM

- DOM: Document Object Model
- interface between javascript and HTML/CSS
- The browser turns every HTML tag into a Javascript object that we can manipulate.
- Everything is stored inside of the `document` object.

## document

```js
- Root Element `<html>`
  - Element `<head>`
    - Element `<title>`
      - Text `...`
  - Element `<body>`
    - Element `<a>`
      - Attribute `href`
      - Text `...`
    - Element `<h1>`
      - Text `...`
```
