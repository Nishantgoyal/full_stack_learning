# Bootstrap buttons

Use the button classes on an `<a>`, `<button>` or `<input>` element.

## Options

- Mandatory class `btn`
- Combine with optional class
  - `btn-default`
  - `btn-primary`
  - `btn-success`
  - `btn-info`
  - `btn-warning`
  - `btn-danger`
  - `btn-link`

## Sizes

- Add `.btn-lg`, `btn-sm` or `btn-xs` for additional sizes
- To create block level buttons, add `.btn-block` class
